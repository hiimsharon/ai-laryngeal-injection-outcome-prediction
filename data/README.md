# Data Directory
本資料夾用於存放本專案所需之資料（data），包含示範用樣本資料、特徵範例、以及資料說明文件。

---

## Data Privacy & Ethics Notice (重要)
由於本研究涉及臨床/病人相關資料，基於病人隱私保護、研究倫理與資料治理規範考量：

- 本 GitHub repository **不提供**任何可識別個人的原始病人資料
  - 原始音檔（wav/mp3）、原始病歷紀錄、可回推個體身分之檔案

---
依照專案公開與可重現性需求，本資料夾包含以下內容

1. `sample/`
   - 少量資料（已去識別化）
   - 用於展示資料格式與程式可正常執行（不包含敏感資訊）

2. `README.md`
   - 本說明文件，描述資料可得性、限制與資料結構
